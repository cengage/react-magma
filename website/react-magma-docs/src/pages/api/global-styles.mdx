<LeadParagraph>
  The GlobalStyles component should be included at the top of your application
  to provide global css, including the required fonts, body styles, and some
  generic styles for links and headings.
</LeadParagraph>

## Usage

The `GlobalStyles` component can be nested inside the <Link to="/api-intro/styles-and-themes/#using_themes">ThemeContext.Provider</Link> to inherit styles from the theme.

```tsx
import React from 'react';

import { GlobalStyles } from 'react-magma-dom';

export function Example() {
  return (
    <>
      <GlobalStyles />
      <div>Styles applied</div>
    </>
  );
}
```
