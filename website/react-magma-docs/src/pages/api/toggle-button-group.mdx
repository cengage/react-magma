<DocsHeading type="api">ToggleButtonGroup</DocsHeading>

The Toggle Button Group is a container for multiple Toggle Buttons and enables controlled behaviors throughout the grouping.

## Basic Usage

The ToggleButtonGroup enables styling and selection states across a group of ToggleButtons.

```typescript
import React from 'react';
import { ToggleButton, ToggleButtonGroup } from 'react-magma-dom';
import { CheckIcon } from 'react-magma-icons';

export function Example() {
  return (
    <ToggleButtonGroup>
      <ToggleButton aria-label="Check icon" icon={<CheckIcon />} />
      <ToggleButton aria-label="Check icon" icon={<CheckIcon />} />
      <ToggleButton aria-label="Check icon" icon={<CheckIcon />} />
      <ToggleButton aria-label="Check icon" icon={<CheckIcon />} />
    </ToggleButtonGroup>
  );
}
```

## No Space

Removes margins and border radius on each button then applies the border-radius around the container.

```typescript

```

## Single Select

Enforces one toggled state across a single select.

```typescript
import React from 'react';

export function Example() {

      <ToggleButton aria-label="Check icon" icon={<CheckIcon />} />
      <ToggleButton aria-label="Check icon" icon={<CheckIcon />} />
    </ToggleButtonGroup>
  );

```

## Default Selection

`defaultChecked` lets a specific button be selected in a ToggleButtonGroup. Once clicked it will deselect and behave like the other buttons accordingly.

```typescript
import React from 'react';
import { ToggleButton, ToggleButtonGroup } from 'react-magma-dom';
import { CheckIcon } from 'react-magma-icons';

export function Example() {
  return (
    <ToggleButtonGroup>
      <ToggleButton aria-label="Check icon" icon={<CheckIcon />} />
      <ToggleButton
        defaultChecked
        aria-label="Check icon"
        icon={<CheckIcon />}
      />
      <ToggleButton aria-label="Check icon" icon={<CheckIcon />} />
      <ToggleButton aria-label="Check icon" icon={<CheckIcon />} />
    </ToggleButtonGroup>
  );
}
```

## Single Selection

If the prop `singleSelect` is used, then toggles will behave like a radio selection.

```typescript
import React from 'react';
import { ButtonSize, ToggleButton, ToggleButtonGroup } from 'react-magma-dom';

export function Example() {

      <ToggleButton aria-label="Check icon" icon={<CheckIcon />} />
      <ToggleButton aria-label="Check icon" icon={<CheckIcon />} />

}
```

## Exclusive Selection

When `exclusive` is enabled, it ensures a radio style ToggleButtonGroup has at least one selected state.

```typescript
import React from 'react';
import { ButtonSize, ToggleButton, ToggleButtonGroup } from 'react-magma-dom';
import {
  FormatAlignLeftIcon,
  FormatAlignCenterIcon,
  FormatAlignRightIcon,
  FormatAlignJustifyIcon,
  SettingsIcon,
} from 'react-magma-icons';

export function Example() {
  return (
    <ToggleButtonGroup noSpace exclusive singleSelect>
      <ToggleButton aria-label="Left align" icon={<FormatAlignLeftIcon />} />
      <ToggleButton
        aria-label="Center align"
        icon={<FormatAlignCenterIcon />}
      />
      <ToggleButton aria-label="Right align" icon={<FormatAlignRightIcon />} />
      <ToggleButton
        aria-label="Justify align"
        icon={<FormatAlignJustifyIcon />}
      />
    </ToggleButtonGroup>
  );
}
```

## Size

Three sizes are configurable with the `ButtonSize` prop between `small`, `medium` (default), and `large` across the group.

```typescript
import React from 'react';
import {
  ButtonSize,
  Container,
  ToggleButton,
  ToggleButtonGroup,
} from 'react-magma-dom';

export function Example() {
  return (
    <Container style={{ padding: '12px' }}>
      <ToggleButtonGroup>
        <ToggleButton size={ButtonSize.small}>Toggle 1</ToggleButton>
        <ToggleButton size={ButtonSize.medium}>Toggle 2</ToggleButton>
        <ToggleButton size={ButtonSize.large}>Toggle 3</ToggleButton>
      </ToggleButtonGroup>
    </Container>
  );
}
```
