<DocsHeading type="api">ToggleButtonGroup</DocsHeading>

The Toggle Button Group is a container for multiple Toggle Buttons and enables controlled behaviors throughout the grouping.

## Basic Usage

The ToggleButtonGroup enables styling with an optional `noSpace` prop, and different selection states `singleSelect` and `requiredSelect`.

```typescript
import React from 'react';
import { ToggleButton, ToggleButtonGroup } from 'react-magma-dom';
import { CheckIcon } from 'react-magma-icons';

export function Example() {
  return (
    <ToggleButtonGroup>
      <ToggleButton aria-label="Check icon" icon={<CheckIcon />} />
      <ToggleButton aria-label="Check icon" icon={<CheckIcon />} />
      <ToggleButton aria-label="Check icon" icon={<CheckIcon />} />
      <ToggleButton aria-label="Check icon" icon={<CheckIcon />} />
    </ToggleButtonGroup>
  );
}
```

## Default Selection

`defaultChecked` lets a specific button be selected in a ToggleButtonGroup. Once clicked it will deselect and behave like the other buttons accordingly.

```typescript
import React from 'react';
import { ToggleButton, ToggleButtonGroup } from 'react-magma-dom';
import { CheckIcon } from 'react-magma-icons';

export function Example() {
  return (
    <ToggleButtonGroup>
      <ToggleButton aria-label="Check icon" icon={<CheckIcon />} />
      <ToggleButton
        defaultChecked
        aria-label="Check icon"
        icon={<CheckIcon />}
      />
      <ToggleButton aria-label="Check icon" icon={<CheckIcon />} />
      <ToggleButton aria-label="Check icon" icon={<CheckIcon />} />
    </ToggleButtonGroup>
  );
}
```

## Single Selection

If the prop `singleSelect` is used, then toggles will behave like a radio selection.

```typescript
import React from 'react';
import { ButtonSize, ToggleButton, ToggleButtonGroup } from 'react-magma-dom';
import { CheckIcon } from 'react-magma-icons';

export function Example() {
  return (
    <ToggleButtonGroup noSpace singleSelect>
      <ToggleButton aria-label="Check icon" icon={<CheckIcon />} />
      <ToggleButton aria-label="Check icon" icon={<CheckIcon />} />
      <ToggleButton aria-label="Check icon" icon={<CheckIcon />} />
      <ToggleButton aria-label="Check icon" icon={<CheckIcon />} />
    </ToggleButtonGroup>
  );
}
```

## Required Selection

When `requiredSelect` is enabled, it ensures a radio style ToggleButtonGroup has at least one selected state.

```typescript
import React from 'react';
import { ButtonSize, ToggleButton, ToggleButtonGroup } from 'react-magma-dom';
import {
  FormatAlignLeftIcon,
  FormatAlignCenterIcon,
  FormatAlignRightIcon,
  FormatAlignJustifyIcon,
  SettingsIcon,
} from 'react-magma-icons';

export function Example() {
  return (
    <ToggleButtonGroup noSpace requiredSelect singleSelect>
      <ToggleButton aria-label="Left align" icon={<FormatAlignLeftIcon />} />
      <ToggleButton
        aria-label="Center align"
        icon={<FormatAlignCenterIcon />}
      />
      <ToggleButton aria-label="Right align" icon={<FormatAlignRightIcon />} />
      <ToggleButton
        aria-label="Justify align"
        icon={<FormatAlignJustifyIcon />}
      />
    </ToggleButtonGroup>
  );
}
```

## Controlled Selection

To require at least one active selection throughout a group, use the `onChange` property on `ToggleButtonGroup`.

```typescript
import React from 'react';
import { ButtonSize, ToggleButton, ToggleButtonGroup } from 'react-magma-dom';
import {
  FormatAlignLeftIcon,
  FormatAlignCenterIcon,
  FormatAlignRightIcon,
  FormatAlignJustifyIcon,
  SettingsIcon,
} from 'react-magma-icons';

export function Example() {
  const [alignment, setAlignment] = React.useState(() => ['left']);

  const handleChange = (
    event: React.MouseEvent<HTMLButtonElement>,
    newAlignment: string[]
  ) => {
    if (newAlignment.length) {
      setAlignment(newAlignment);
    }
  };

  return (
    <ToggleButtonGroup
      requiredSelect
      onChange={handleChange}
      noSpace
      value={alignment}
    >
      <ToggleButton
        aria-label="Left align"
        icon={<FormatAlignLeftIcon />}
        value="left"
      />
      <ToggleButton
        aria-label="Center align"
        icon={<FormatAlignCenterIcon />}
        value="center"
      />
      <ToggleButton
        aria-label="Right align"
        icon={<FormatAlignRightIcon />}
        value="right"
      />
      <ToggleButton
        aria-label="Justify align"
        icon={<FormatAlignJustifyIcon />}
        value="justify"
      />
    </ToggleButtonGroup>
  );
}
```

## Size

Three sizes are configurable with the `ButtonSize` prop between `small`, `medium` (default), and `large` across the group.

```typescript
import React from 'react';
import {
  ButtonSize,
  Container,
  ToggleButton,
  ToggleButtonGroup,
} from 'react-magma-dom';

export function Example() {
  return (
    <Container style={{ padding: '12px' }}>
      <ToggleButtonGroup>
        <ToggleButton size={ButtonSize.small}>Toggle 1</ToggleButton>
        <ToggleButton size={ButtonSize.medium}>Toggle 2</ToggleButton>
        <ToggleButton size={ButtonSize.large}>Toggle 3</ToggleButton>
      </ToggleButtonGroup>
    </Container>
  );
}
```
