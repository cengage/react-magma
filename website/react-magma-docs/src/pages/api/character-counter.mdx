<DocsHeading type="api">Character Counter</DocsHeading>

A numeric counter that is used by <Link to="/api/input">Input</Link> and <Link to="/api/textarea">Textarea</Link> below either which dynamically increments or deducts a value based on a set length.

## Basic Usage

The Character Counter is called when the prop of `maxLength` is set within an <Link to="/api/input">Input</Link> or a <Link to="/api/textarea">Textarea</Link>.

As the user types, a counter is displayed below the Input or Textarea counting down to the allowable length. Once this is exceeded, an error message shows in place counting the overrage of characters.

```typescript
import React from 'react';
import { Input } from 'react-magma-dom';

export function Example() {
  return <Input maxLength={4} labelText="Default Character Counter" />;
}
```

## Inverse

The `isInverse` prop is a boolean meant for interfaces using dark backgrounds.

```typescript
import React from 'react';
import { Container, Input } from 'react-magma-dom';

export function Example() {
  return (
    <Container isInverse style={{ padding: '12px' }}>
      <Input isInverse maxLength={4} labelText="Character Counter" />
    </Container>
  );
}
```

## Internationalization

If you're using strings in your component, they must be internationalized.

Full example of <Link to="/api/internationalization">internationalization override options</Link>

```tsx
import React from 'react';
import { es } from 'date-fns/locale';
import { Input, I18nContext, defaultI18n } from 'react-magma-dom';

export function Example() {
  return (
    <I18nContext.Provider
      value={{
        ...defaultI18n,
        example: 'example i18n',
      }}
    >
      <Input maxLength={4} labelText="Character Counter" />
    </I18nContext.Provider>
  );
}
```
