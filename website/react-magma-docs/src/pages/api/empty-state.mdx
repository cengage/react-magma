<PageContent componentName="empty_state" type="api">

<LeadParagraph>
  The EmptyState component communicates a lack of content and guides users
  towards action. It supports an illustration, title, body text, up to three
  action buttons, and a content slot for custom content.
</LeadParagraph>

## Basic Usage

```tsx
import React from 'react';

import { EmptyState } from 'react-magma-dom';
import { SearchIcon } from 'react-magma-icons';

export function Example() {
  return (
    <EmptyState
      illustration={<SearchIcon />}
      title="No results found"
      body="Try adjusting your search or filter criteria to find the courses you are looking for."
      primaryAction={{
        label: 'Search Again',
        onClick: () => console.log('Search clicked'),
      }}
      secondaryAction={{
        label: 'Clear Filters',
        onClick: () => console.log('Clear clicked'),
      }}
      tertiaryAction={{
        label: 'Browse All Courses',
        onClick: () => console.log('Browse clicked'),
      }}
    />
  );
}
```

## Tertiary Action

The tertiary action renders as a link-style button below the primary and secondary buttons. Use it for a lower-priority alternative action.

```tsx
import React from 'react';

import { EmptyState } from 'react-magma-dom';
import { SearchIcon } from 'react-magma-icons';

export function Example() {
  return (
    <EmptyState
      illustration={<SearchIcon />}
      title="No courses found"
      body="We couldn't find any courses matching your search. Try browsing the full catalog."
      tertiaryAction={{
        label: 'Browse Course Catalog',
        onClick: () => console.log('Browse clicked'),
      }}
    />
  );
}
```

## Content Slot

Pass children to render custom content between the text and action buttons. This is useful for inline search fields, filter chips, or other interactive elements.

```tsx
import React from 'react';

import { EmptyState, Search } from 'react-magma-dom';
import { SearchIcon } from 'react-magma-icons';

export function Example() {
  return (
    <EmptyState
      illustration={<SearchIcon />}
      title="No results found"
      body="Try a different search term or browse by category."
    >
      <Search
        isClearable
        placeholder="Search courses..."
        onSearch={value => console.log('Search:', value)}
      />
    </EmptyState>
  );
}
```

## Danger Mode

Use the `isDanger` prop to display an error state with danger colors on the illustration. The buttons remain the default primary color for consistency.

```tsx
import React from 'react';

import { EmptyState } from 'react-magma-dom';
import { ErrorIcon } from 'react-magma-icons';

export function Example() {
  return (
    <EmptyState
      illustration={<ErrorIcon />}
      isDanger
      title="Unable to load assignments"
      body="We encountered an error while loading your course content. Please try again or contact support."
      primaryAction={{
        label: 'Try Again',
        onClick: () => console.log('Retry clicked'),
      }}
      secondaryAction={{
        label: 'Contact Support',
        onClick: () => console.log('Contact clicked'),
      }}
    />
  );
}
```

## Without Illustration

The illustration is optional. You can render an EmptyState with just text and actions.

```tsx
import React from 'react';

import { EmptyState } from 'react-magma-dom';

export function Example() {
  return (
    <EmptyState
      title="No submissions yet"
      body="Students have not submitted any assignments for this course yet. Check back after the due date."
      primaryAction={{
        label: 'View Assignment Details',
        onClick: () => console.log('View clicked'),
      }}
    />
  );
}
```

## Without Buttons

Action buttons are also optional. You can display an informational empty state without any actions.

```tsx
import React from 'react';

import { EmptyState } from 'react-magma-dom';
import { NotificationsIcon } from 'react-magma-icons';

export function Example() {
  return (
    <EmptyState
      illustration={<NotificationsIcon />}
      title="No upcoming deadlines"
      body="You're all caught up! There are no assignments or quizzes due this week."
    />
  );
}
```

## Minimal

At minimum, you can render an EmptyState with just a title.

```tsx
import React from 'react';

import { EmptyState } from 'react-magma-dom';

export function Example() {
  return <EmptyState title="No courses enrolled" />;
}
```

## Inverse

The `isInverse` prop renders the component with colors suitable for dark backgrounds.

```tsx
import React from 'react';

import { EmptyState, Card, CardBody, magma } from 'react-magma-dom';
import { SearchIcon } from 'react-magma-icons';

export function Example() {
  return (
    <Card background={magma.colors.primary600} isInverse>
      <CardBody>
        <EmptyState
          illustration={<SearchIcon />}
          isInverse
          title="No results found"
          body="Try adjusting your search or filter criteria to find the courses you are looking for."
          primaryAction={{
            label: 'Search Again',
            onClick: () => console.log('Search clicked'),
          }}
          secondaryAction={{
            label: 'Clear Filters',
            onClick: () => console.log('Clear clicked'),
          }}
          tertiaryAction={{
            label: 'Browse All Courses',
            onClick: () => console.log('Browse clicked'),
          }}
        />
      </CardBody>
    </Card>
  );
}
```

## Loading State

Use the `isLoading` prop to show a loading spinner instead of the content. This is useful when data is being fetched.

```tsx
import React from 'react';

import { EmptyState } from 'react-magma-dom';

export function Example() {
  return <EmptyState isLoading />;
}
```

The loading spinner color adapts to `isDanger` and `isInverse` props.

## Custom Children

You can pass custom content as children to the EmptyState component. Here, clickable `Tag` components let users browse courses by category.

```tsx
import React from 'react';

import { EmptyState, Tag } from 'react-magma-dom';
import { SearchIcon } from 'react-magma-icons';

export function Example() {
  return (
    <EmptyState
      illustration={<SearchIcon />}
      title="No courses found"
      body="Try browsing by category instead."
      tertiaryAction={{
        label: 'Browse All Courses',
        onClick: () => console.log('Browse clicked'),
      }}
    >
      <div style={{ display: 'flex', gap: '8px', flexWrap: 'wrap', justifyContent: 'center' }}>
        <Tag onClick={() => console.log('Biology')}>Biology</Tag>
        <Tag onClick={() => console.log('Chemistry')}>Chemistry</Tag>
        <Tag onClick={() => console.log('Physics')}>Physics</Tag>
        <Tag onClick={() => console.log('Mathematics')}>Mathematics</Tag>
        <Tag onClick={() => console.log('Computer Science')}>Computer Science</Tag>
      </div>
    </EmptyState>
  );
}
```
