## Installation

<Alert variant="warning"><strong>Important!</strong> CarbonChart is imported from `@react-magma/charts` which must be installed as a peer dependency.<br/>
In addition, you must manually import the styles:
`import '@react-magma/charts/dist/charts.css';`</Alert>

### Setup Requirements

Your project needs to be running **React v18**

### Install the Charts Library

Install the React Magma Charts library with `npm`.

```sh-title-noRender-noCodeSandbox
npm install --save react-magma@charts
```

Install peer dependencies

```sh-title-noRender-noCodeSandbox
npm install --save \
    "@emotion/react@^11.14.0" \
    "@emotion/styled@^11.14.0" \
    "react@^18.3.1" \
    "react-dom@^18.3.1" \
    "react-magma-dom@^5.0.0" \
    "react-magma-icons@^3.2.0" \
    "@react-magma/charts@^12.0.0"
```
