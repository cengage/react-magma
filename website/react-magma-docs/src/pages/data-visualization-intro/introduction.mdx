## Installation

### Setup Requirements

Your project needs to be running **React v17** or greater

### Install the Charts Library

Install the React Magma Charts library with `npm`.

```sh title noRender noCodeSandbox
npm install --save react-magma@charts
```

Install peer dependencies

```sh title noRender noCodeSandbox
npm install --save \
    "@emotion/react@^11.13.0" \
    "@emotion/styled@^11.13.0" \
    "react@^17.0.2" \
    "react-dom@^17.0.2" \
    "react-magma-dom@^4.7.0-next.1" \
    "react-magma-icons@^3.0.0"
```
